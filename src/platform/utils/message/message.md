# 消息工具 (message/)

## 概述

消息工具提供消息格式化、验证和任务委托书处理功能。

## 模块列表

### message_formatter.js
- **职责**：消息格式化
- **功能**：
  - 格式化消息为标准格式
  - 消息内容转换
  - 消息模板渲染

### message_validator.js
- **职责**：消息验证
- **功能**：
  - 验证消息格式
  - 验证必填字段
  - 验证字段类型和值

### task_brief.js
- **职责**：任务委托书处理
- **功能**：
  - 验证任务委托书格式
  - 格式化任务委托书
  - 提取任务信息

## 依赖关系

- 三个模块相对独立
- 可能共享一些常量定义

## 注意事项

- 消息格式应该有明确的规范
- 验证应该严格但不过度
- 任务委托书应该支持扩展
