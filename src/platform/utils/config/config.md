# 配置工具 (config/)

## 概述

配置工具提供配置加载和配置服务功能。

## 模块列表

### config_loader.js
- **职责**：配置加载（从 config.js 重命名）
- **功能**：
  - 从文件加载配置
  - 配置合并
  - 配置验证
  - 环境变量替换

### config_service.js
- **职责**：配置服务
- **功能**：
  - 提供配置查询接口
  - 配置热更新
  - 配置监听
  - 配置缓存

## 依赖关系

- config_service.js 依赖 config_loader.js
- config_loader.js 依赖文件系统

## 注意事项

- 配置加载应该支持多种格式（JSON、YAML 等）
- 配置应该支持环境变量
- 配置服务应该支持热更新
- 配置验证应该在加载时进行
