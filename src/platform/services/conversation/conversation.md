# 会话服务 (conversation/)

## 概述

会话服务负责智能体的对话历史管理、上下文压缩和 Token 统计。

## 模块列表

### conversation_manager.js
- **职责**：会话管理器
- **功能**：
  - 上下文压缩
  - Token 使用统计
  - 对话历史持久化
  - 上下文状态检查
  - 对话历史查询

## 依赖关系

- 依赖文件系统进行持久化
- 依赖 LLM 服务进行 Token 计数

## 注意事项

- 上下文压缩应该保留关键信息
- Token 统计应该准确
- 对话历史应该支持增量更新
