## 新增功能说明

### 资源管理功能
模块新增了页面资源管理功能，包括：

1. **获取资源列表** (chrome_get_resources)
   - 获取页面上的各类资源（图片、CSS、JavaScript、视频、音频等）
   - 支持过滤资源类型
   - 支持包含或排除 data URL
   - 返回资源的 URL、类型、尺寸等详细信息

2. **保存资源到工件** (chrome_save_resource)
   - 将页面资源保存到工件系统
   - 支持普通 URL 和 data URL
   - 自动识别资源格式（png、jpg、gif、webp等）
   - 返回工件ID，可在工件管理器中查看
   - 保存时记录来源页面信息（URL、标题等）

### 使用场景
- 批量下载页面图片
- 保存页面资源用于离线分析
- 收集网页素材
- 页面资源审计
- 内容归档
