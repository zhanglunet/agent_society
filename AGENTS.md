# 项目

这是一个基于狮偶编程语言的商业项目
这个项目的名字叫OpenBlock，底层的技术是狮偶（RoarLang）。
在server\static\openblock包含了狮偶的全部代码，包括编辑器、编译器、翻译器、解释器等。
在server\static\testing中包含了狮偶官方面向非程序员初学者的样例库。
其他部分都是OpenBlock的组成部分。

# 启动项目

通常在编辑网页端内容的时候，服务器是我已经手工启动并保持运行状态的，你不需要启动服务器。
如果有必要启动的话，在使用node启动 server\index.mjs 。

# 环境

我的系统是Windows11，命令行是PowerShell
生产环境是CentOS8
